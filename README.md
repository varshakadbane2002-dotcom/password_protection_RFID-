# password_protection_RFID-
Two-step authentication system The project uses an EM-18 RFID reader to first verify a registered RFID card, then
asks the user to enter a 4-digit password, adding an extra layer of security.
– A controller checks both the RFID card ID and the entered password with stored data only if both are correct, the
system unlocks or grants access.
– Used for secure doors, lockers, or restricted areas, where valid RFID + correct password results in ’Access Granted,’
otherwise ’Access Denied’.
